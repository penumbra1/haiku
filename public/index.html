A tiny classic haiku JSON database

Welcome!

This is a collection of selected haikus by three Japanese pioneers of the genre: Kobayashi Issa, Matsuo Basho, and Yosa Buson.

All rights to the texts belong to the authors and the translators, and I put up this little database for a study project. Please use it with discretion!
